# pradipkunwar.github.io

Personal academic portfolio website for Pradip Kunwar.

## Deploy to GitHub Pages

```bash
git clone https://github.com/pradipkunwar/pradipkunwar.github.io.git
cp -R pradipkunwar.github.io/* ./pradipkunwar.github.io/
cd pradipkunwar.github.io
git add .
git commit -m "Build personal portfolio website"
git push origin main
```

GitHub Pages will serve the site from `https://pradipkunwar.github.io`.
